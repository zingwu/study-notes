* 解析
** moment()
   获取当前日期和时间，只需调用不带参数的 moment() 即可。

* 操作
** subtract()
   通过减去时间来改变原始的 moment。 moment().subtract(number,string)

例如：获取昨天：moment().substract(1,'days')
